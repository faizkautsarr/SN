
## Installation

To Install this project run

```bash
  npm install
```

After Install success
```bash
  npm start
```

By Default you can run it at
```bash
  http://localhost:9999/notifications
```
Or if you want using your own api, please follow this sample structure: [Here](http://13.239.117.240:10077/api/bpjs/notification/list)
```bash
  http://localhost:9999/notifications?apiurl={INSERT_YOUR_API_HERE}
```

For ready to demo prototype I already deployed it, access it on [http://13.239.117.240:9999/notifications?apiurl=http://13.239.117.240:10077/api/bpjs/notification/list](http://13.239.117.240:9999/notifications?apiurl=http://13.239.117.240:10077/api/bpjs/notification/list)
